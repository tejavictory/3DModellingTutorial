# 3DModellingTutorial
WebApps 
Required:
Autodesk MAYA
Three.js libraries
OBJ and MTL loader
Orbit Controls

Implementation:
1. Create a 3D model in MAYA
2. Export the model in OBJ format
3. Load the model in three.js
4. Run the server.
